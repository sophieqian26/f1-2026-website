# F1 2026 Season Hub

A static Formula 1 2026 season website with live schedule data, results, standings, driver news, race cards, profile cards, and odds-focused race preview UI.

## GitHub Pages

This project is ready to publish from the repository root with GitHub Pages.

Recommended Pages settings:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`

After publishing, the site URL will look like:

`https://YOUR-USERNAME.github.io/YOUR-REPOSITORY-NAME/`
